# ggezCTF_workshop

## How to run it 
# Frontend 
- cd frontend
- npm run build

# Backend
- cd backend
- python3 -m venv .venv
- source .venv/bin/activate
- pip install -r requirement.txt
- python3 app.py

it will have the warning if you don't want to see it 
- use waitress
